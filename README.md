# online-shop

## Setup

Type this in terminal to install the dependencies:

```bash
npm install
```

## Development Server

Start the development server on http://localhost:8080

```bash
npm run dev
```

## Used on this project
<ul>
  <li>Node Js</li>
  <li>Express</li>  
  <li>Prisma</li>      
  <li>SQLite Database</li> 
</ul>
